# Funcionalidades Y Valor Para El Usuario

## 1) Objetivos del producto para usuario final

La app cubre objetivos concretos:

- Capturar tareas sin friccion.
- Priorizar accion diaria.
- Reducir procrastinacion sostenida.
- Mejorar adherencia por narrativa y recompensas.
- Dar visibilidad semanal del rendimiento.

## 2) Funcionalidades clave (vista usuario)

### A) Diario de misiones

- Escribes en lenguaje natural.
- S.A.M. transforma el input en una gesta epica.
- Puedes cerrar cada mision como exito o fracaso.

### B) Barra de viaje al Monte del Destino

- Sustituye el concepto de boss HP por avance narrativo de travesia.
- El usuario ve progreso restante en tiempo real tras cierre de misiones.
- HUD visible solo en Diario para no ensuciar otras vistas.

### C) Sistema de oscuridad progresiva

- Si fallas, la atmosfera visual se oscurece por niveles.
- Cada dia con tareas fallidas aumenta presion visual.
- Mecanica pensada para reforzar urgencia sin castigo destructivo.

### D) Palantir (IA predictiva)

- Estima riesgo y recomienda accion.
- Evita respuestas alarmistas para nuevos usuarios sin historial.
- Mantiene cache por usuario para no cruzar estados.

### E) Perfil RPG

- Raza, nivel, titulo evolutivo, oro y logros.
- Cambio de raza premium con coste alto de oro (decision rara, no casual).

### F) Inventario + Forja

- Loot por exito.
- Etiqueta NEW por usuario hasta revisar detalle.
- Fusion/forja de objetos para progresion extendida.

### G) Mapa tactico semanal

- 4 graficas por raza.
- Recuento semanal de cumplidas/fallidas.
- Reinicio semanal anunciado: domingo 23:59.

## 3) Resultado esperado en comportamiento

- Mas ejecucion diaria (menos friccion).
- Menos abandono prolongado (feedback visual + narrativa).
- Mayor sensacion de control y progreso.
- Mayor retorno diario por sistema de recompensas.

## 4) Indicadores sugeridos para defensa

- Frecuencia de uso diario.
- Tasa de tareas completadas.
- Tiempo medio de cierre desde creacion.
- Evolucion de rachas semanales.
- Distribucion de uso por modulos (diario, perfil, mapa, forja).
